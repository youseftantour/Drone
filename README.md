# Autonomous Drone Navigation System

## Overview

This project was completed as part of CIS 492/593 Embedded Systems at Cleveland State University. The objective was to design, build, program, and test an Air:Bit 2 drone capable of autonomously navigating to another team's rover, capturing an image, and returning safely.

The project combined embedded systems programming, hardware assembly, wireless communication, flight control, and system testing. The drone was programmed using Microsoft MakeCode and JavaScript running on BBC Micro:bit controllers.

## Project Goals

* Build and configure a functional Air:Bit 2 drone
* Develop software for drone flight control
* Integrate a camera module for image capture
* Achieve stable and reliable flight
* Navigate to a target location and capture an image
* Test and improve system performance through iterative development

## Technical Challenges

During development, the team encountered several challenges:

* Limited battery life during testing
* Loose motor wiring affecting thrust generation
* Excess weight reducing lift capability
* Flight instability and sensitivity issues
* Camera integration and wiring complexity
* Hardware damage during collision testing

These issues were addressed through troubleshooting, hardware adjustments, software tuning, and repeated testing.
